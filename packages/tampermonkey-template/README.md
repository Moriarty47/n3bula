# `@n3bula/tampermonkey-template`

### A CLI tool for creating tampermonkey template.

## Install

```sh
npm install @n3bula/tampermonkey-template
# or
yarn add @n3bula/tampermonkey-template
# or
pnpm add @n3bula/tampermonkey-template
```
